Relative Valuation Model Generator

The Relative Valuation Model Generator allows users to compare companies on the basis of their financials. More specifically, this software gathers data from income statements, balance sheets, and cash flow statements over the last 4 years. It will gather the data of up to 7 companies and plot up to 5 statstics. Both the raw data as well as the derivatives of the data are plotted in order to allow the comparison of growth rates over time. Below are a few examples of the output of the Relative Valuation Model Generator comparing companies in the same sector. This program was written in Python using the following APIs: pandas, yfinance, matplotlib, PySimpleGUI, PyQt5.
